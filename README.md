# PromptFlow Client

Frontend starter for the PromptFlow assignment.

## Tech Stack

- Next.js
- JavaScript
- Tailwind CSS

## Environment Variables

Create a local environment file from the example and use:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000
```

## Available Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
